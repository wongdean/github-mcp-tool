# 🎯 Model Context Protocol (MCP) 技术分享

## 📋 Presentation 大纲

### 第一部分：MCP 基础概念 (10分钟)

#### 1.1 什么是MCP？
- **定义**：Model Context Protocol - 模型上下文协议
- **作用**：连接AI模型与外部工具/数据源的标准化协议
- **价值**：让AI从"只会聊天"升级为"能实际操作"

#### 1.2 传统AI vs MCP增强AI
```
传统AI：只能基于训练数据回答
MCP增强AI：能实时获取最新数据并执行操作
```

#### 1.3 MCP核心概念
- **Tools（工具）**：执行具体功能
- **Resources（资源）**：提供数据内容
- **Prompts（模板）**：专业指导框架
